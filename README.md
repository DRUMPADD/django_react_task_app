First, clone this repository using the next command on your terminal:
```bash
    git clone https://github.com/DRUMPADD/django_react_task_app.git
```

## Execute Django server
```bash
    cd backend
    manage.py runserver
```

## Create user
```bash
    manage.py createsuperuser
```

## Make migrations
```bash
    manage.py makemigrations
    manage.py migrate
```


## Execute react app
```bash
    cd client
    npm run dev
```
