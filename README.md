## Execute server
```bash
    cd backend
    manage.py runserver
```

## Create user
```bash
    manage.py createsuperuser
```

## Make migrations
```bash
    manage.py makemigrations
    manage.py migrate
```


## Execute client
```bash
    cd client
    npm run dev
```
